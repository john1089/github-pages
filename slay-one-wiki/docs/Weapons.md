# 武器
[Slay.one](Slay-one.md) 中的**武器**是战斗中用来对敌人造成伤害的物品。每种武器都有自己的优点和缺点。武器作为[可捡起物品](Pickups.md)在地图上被发现。武器需要使用弹药。

有几种类型的武器。下表描述了它们的差异。
